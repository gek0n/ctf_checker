Description:
Эта программа проверяет главную страницу сайта ctftime.org на наличие изменений в предстоящих играх. В случае появления новой игры, изменения времени игры либо подтверждении участия новыми командами, программа сообщает об этом при помощи записанного заранее звукового обращения. Также при появлении новой игры или изменении времени проведения на экран выводится всплывающее окно с соответствующим текстом. Для отслеживания изменений данные, считанные с сайта, помещаются в файл upcoming_ctf.txt для сравнения с текущим состоянием при очередном запуске.
Основное предназначение программы - работа в качестве сервиса. Файл start.bat делает запись в планировщике заданий для windows: программа будет запускаться каждый час для проверки состояния игр. Соответственно файл stop.bat убирает эту запись из планировщика заданий.

Installation:
1) Скопируйте папку со всем содержимым в место, откуда разрешено выполнение исполняемых файлов (и откуда вы случайно не удалите её).
2) Запустите файл start.bat

(Для однократного использования просто дважды щелкните на файл ctf_checker.pyw)

System required:
Любая система с установленным Python 2.7
(При использовании системы, отличной от Windows прописывать прорамму в планировщик щаданий придется вручную)

Modification:
При желании можно заменить файлы в папке wav_phrases на любые другие .wav файлы методом замены с указанием идентичного имени.
